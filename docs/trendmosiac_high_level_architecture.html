TrendMosaic is a multi-source trend intelligence tool that surfaces emerging technologies from real-world discussions. It aggregates content from Reddit, Hacker News, and Stack Overflow, using LLMs to distill insights and identify themes, sentiment, and tools at the heart of the conversation.

üß± Architecture Overview

  1. Data Ingestion
    • Sources: Reddit (via PRAW), Hacker News, Stack Overflow (via StackExchange API)
    • Topic-based keyword filtering
    • Modular, extensible fetch layer per source
  2. LLM Processing & Enrichment
    • Summarization using GPT-3.5
    • Sentiment classification
    • Tag extraction (emerging tools, platforms, themes)
    • Optional future clustering / deduplication
  3. UI & Interaction Layer
    • Streamlit frontend for rapid exploration
    • Topic search + tag filtering
    • Interactive Q&A with context-aware GPT answers
  4. Data Storage (coming soon)
    • JSON export for MVP
    • Planned: PostgreSQL or SQLite for structured metadata
    • Planned: Chroma or FAISS for semantic search with vector embeddings
