⚽ Underdogged

A lightweight football match prediction engine that compares machine learning probabilities with bookmaker odds.

What does it do?

  • Fetches upcoming Premier League fixtures and recent match history
  • Runs a simple ML model to predict match outcomes with confidence levels
  • Highlights potential “underdog” mismatches using probability thresholds
  • Includes head-to-head stats and recent team performance comparisons

⚙️ Underdogged – Features

  • 📅 Fixture Integration
    Automatically updates with new fixtures from a live data feed.
  • 🔮 Model Confidence Filters
    Filter by predicted win probability to surface surprising matchups.
  • 🤝 Head-to-Head Breakdown
    View recent results between any two selected teams.
  • 📊 Team Form Visualisation
    See how each team has performed in their last five games.
