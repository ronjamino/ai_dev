Underdogged is a lightweight football analytics tool that surfaces potential mismatches between machine learning predictions and bookmaker odds. It provides head-to-head analysis, recent team form, and model-backed match outcome predictions in a user-friendly web app.

ğŸ§± Architecture Overview

  1. Data Ingestion
    • Fixtures scraped from a free football data source
    • Recent match results (for form and head-to-head analysis)
    • Scripted via `fetch_fixtures.py` and scheduled via manual trigger
  2. Data Preparation
    • Feature engineering: form stats, goals, venue encoding
    • Label encoding of match outcomes
    • All stored in a `training_data.csv` for reuse
  3. Model Pipeline
    • Scikit-learn classifier (e.g. LogisticRegression or RandomForest)
    • Trained on historical Premier League data
    • Outputs `predicted_result`, `predicted_proba`, and `confidence_label`
  4. UI & Interaction Layer
    • Streamlit frontend with filters and dropdowns
    • Refresh button triggers re-run of full prediction pipeline
    • Head-to-head + recent form shown side-by-side
