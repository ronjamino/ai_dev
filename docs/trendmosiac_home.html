TrendMosaic

Your AI-powered radar for tracking emerging tech trends across the internet.

What does it do?

  • Aggregates discussions from Reddit, Hacker News, and more
  • Uses LLMs to summarise sentiment, topics, and emerging tools
  • Highlights shifts in community opinions over time

🌟 TrendMosaic – Key Features

  • 🔍 Search by Topic
    Explore discussions across Reddit, Hacker News, and Stack Overflow for any tech keyword (e.g. dbt , DuckDB , data pipelines ).
  • 🧠 AI-Generated Summaries
    Each post is automatically summarised and annotated with key insights using GPT-3.5.
  • 🏷 Auto-Extracted Tags
    Trending tools and concepts (like Snowflake , Airflow , Dagster ) are automatically identified and tagged.
  • 🎯 Tag-Based Filtering
    Focus on what matters by filtering results with a multi-select tag filter in the sidebar.
  • 💬 Context-Aware Q&A
    Ask your own questions about the topic and get a GPT-powered synthesis based on the latest community chatter.
  • 📚 Multi-Source Aggregation
    Pulls from multiple real-time sources:
    • Reddit (e.g. r/dataengineering, r/datascience)
    • Hacker News
    • Stack Overflow
  • 🧩 Expandable Post Cards
    View the summary, sentiment, tags, and source link for each post in a clean, collapsible format.
